# Custom Linux Environment & Automation Scripts

Welcome to my personal systems deployment and configuration repository. This repo showcases my hands-on experience with operating system customization, terminal-based workflows, and automation scripting.

## 🛠️ Environment Overview
* **OS:** Arch Linux (Minimal, keyboard-driven setup)
* **Window Manager:** dwm (Suckless dynamic window manager compiled from source C)
* **Primary Text Editor:** Neovim / Vim (Optimized CLI workflow)
* **Terminal Multiplexer:** Tmux (Advanced window/pane management)

## 📁 Repository Structure
* `/suckless/` - Custom C source code modifications and source compilations for `dwm`, `st`, and `dmenu`.
* `/.config/` - Custom terminal environment configurations (Neovim, Tmux, Picom, Ranger).
* `/scripts/` - Custom Bash scripts engineered for automation.

## 🚀 Key Automation Scripts Highlighted
1. **`rclone_sync.sh`**: Automates localized data backups and directories synchronization with remote cloud environments.
2. **`startup_tmux.sh`**: Scripted terminal layout initializations to automate workspace setups on boot.
3. **`status.sh`**: Lightweight system resources monitoring engine feeding real-time metrics back to the desktop status bar.

## 🎯 Technical Competencies Demonstrated
* Advanced Linux CLI & Core Utilities
* System Level Automation (Bash Shell Scripting)
* Version Control (Git/GitHub)
* Performance Optimization (Zero heavy IDEs, strictly lightweight CLI tools)
